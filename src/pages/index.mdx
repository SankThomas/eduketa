# Welcome to Eduketa

Eduketa is an online educational platform where you can teach yourself how to code using examples and detailed explanations from instructors.

Currently, I am building modules for web development, namely HTML, CSS, JavaScript and an additional ReactJs. More frameworks will be added soon.
